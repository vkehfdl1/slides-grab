Mobile Strategy HTML Templates

Templates use common type names from packs/common-types.json.
Shared tokens in theme.css, shared layout rules in base.css.
Add a theme class on <body> when needed, e.g. <body class="theme-purple-dusk">

Template list (11):
- cover.html: large title cover slide
- contents.html: agenda / outline slide
- section-divider.html: section divider with dark gradient background
- content.html: left-aligned title + paragraph intro
- principles.html: three equal content blocks
- highlight.html: insight highlight slide
- split-layout.html: left text + right image placeholders
- two-columns.html: problem / solution detail slide
- image-text.html: central device mockup + side explanations
- image-description.html: device-focused visual slide
- key-metrics.html: metrics / modules grid slide

Canvas size: 720pt x 405pt (16:9).
